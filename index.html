A Semantic Book Recommender using LLMs

I developed a Semantic Book Recommender System that enables users to discover books by simply describing what they‚Äôre looking for in natural language (e.g., "a suspenseful journey of self-discovery"). The system uses OpenAI's embedding models via LangChain to encode book descriptions into vector representations and stores them in a Chroma vector database.

Users interact through an intuitive Gradio UI, with optional filters for genre and emotional tone (e.g., happy, sad, surprising). The app ranks and returns the most semantically relevant book recommendations, even for abstract or subjective queries.

The entire project is hosted on Hugging Face Spaces, making it publicly accessible for demo and evaluation.
